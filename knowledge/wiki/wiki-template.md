# wiki-template.md

わからない用語が出てきたら、このテンプレートをClaudeに貼って使う。
出力結果をそのまま `Knowledge/wiki/wiki.md` に追記する。

---

## 使い方

1. 下記プロンプトの `[用語名]` を調べたい言葉に置き換える
2. 出力されたmdブロックを `Knowledge/wiki/wiki.md` の該当カテゴリに追記する

---

## プロンプト
- `[用語名]` をwiki.mdに追加し、下記のフォーマットでmdを作成する
- 専門知識がないデザイナーでも読んで理解できる日本語で書く
- 例え話は日常的なものを使う
- コマンドがない概念用語の場合は省略する

```
### [用語名]
一言で説明（例え話や身近なものに置き換えて）

- **なぜ必要か**
  なぜ必要か列挙

- **インストール**（必要な場合のみ）：
  ```
  コマンド
  ```

- **使い方**（必要な場合のみ）：
  ```
  コマンド
  ```

- **関連**：
  関連する用語や技術を列挙
```

---

## 出力例

```
## Homebrew
macOSのパッケージ管理ツール。iPhoneのApp Storeに相当するもの。
開発ツールをコマンド一行でインストール・アップデート・削除できる。

- **なぜ必要か**
  Node.jsなどApple標準外のツールを手軽にインストールするための土台

- **インストール**：
  ```zsh
  /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
  ```

- **使い方**：
  ```zsh
  brew install ツール名   # インストール
  brew update            # Homebrew自体を更新
  brew upgrade           # インストール済みツールを更新
  ```

- **関連**
  Node.js / npm / Claude Code のインストールに必要
```
