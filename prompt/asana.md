# asana.md 
Asanaのタスク概要欄に入れる説明文を、Claudeに作成してもらうためのテンプレート。
誰が読んでも理解できる形にするために使う。


## 使い方
- このmdを用いて、作りたいタスクを作成。
- 出力結果をAsana MCPを利用し、AsanaとGitHubに反映する。
- 親タスクは「#」から始める。サブタスク1は「##」から始める。サブタスク2は「###」から始める。
- 各タスク（#、##、###）の直後に、以下の「タスク概要の形式」をそれぞれ記述すること。

## タスクの基本構成
```
親タスク
└─ サブタスク１　　　#タスクプロセス　
　　└─ サブタスク２　#プロセスのタスク ここからクリアしていく形　
```

## 各タスク概要の形式
```
このプロジェクトが存在する理由・解決する問題を入力
なぜこのタスクを行うのか、どんな課題を解決するのか

**タスク構造**
下層タスクの構造をファイル構造ツリーで表示

親タスク
└─ サブタスク１（#タスクプロセス）
　　└─ サブタスク２（#プロセスのタスク ここからクリアしていく形）

**参照**
GitHubなどのURLを入力

**完了条件**
- タスク完了条件を入力
- タスク完了条件を入力
- タスク完了条件を入力

**用語・前提知識**
今回使う特殊な言葉や、知っておくべき前提ルール
```
