# faq-tem.md
クライアントからの実際の問い合わせ・質問と回答を `00_Global/faq.md` に追記するためのプロンプトテンプレート。
運用マニュアルとは異なり「実際に起きた質問」を時系列で蓄積する。
スタイルルールは `prompt/md.md` に従うこと。


## 使い方
- クライアントからメールで質問が来たら、下記プロンプトをClaudeに貼り付ける
- メール本文をそのまま【入力内容】に貼り付ける
- 出力結果を `00_Global/faq.md` の先頭（最新が上）に追記する


## プロンプト本文

```
以下のメール内容をもとに、faq.mdに追記するQ&Aエントリを作成すること。

【ルール】
・時系列蓄積型で記録する（最新が先頭）
・質問は原文のまま、または自然な日本語に整える程度に留める
・回答が未確定の場合は「対応中」と記載し、後から更新する
・カテゴリタグは以下から最も近いものを選ぶ。該当なければ`#その他`とする
　`#納品` `#修正` `#費用` `#スケジュール` `#CMS操作` `#その他`
・出力はQ&Aエントリのブロックのみ。説明や補足は不要

【出力形式】
### YYYY-MM-DD
タグ：#カテゴリ
頻度：1回目
Q：[質問内容]
A：[回答内容（未確定の場合は「対応中」）]
出典：メール（[クライアント名または会社名]）

【入力内容】
[ここにメール本文を貼り付ける]
```


## faq.mdの構成（追記先ファイルのフォーマット）

```markdown
# faq.md
クライアントからの実際の問い合わせ・質問と回答を蓄積するファイル。
運用マニュアルとは異なり、「実際に起きた質問」を記録する場所。


## 運用ルール
- 質問が来たら日付・カテゴリタグ・質問・回答を記録する
- 回答が決まっていない場合は「対応中」と記載し、後から更新する
- 同じ質問が複数回来たら「頻度」を更新する（再回答の手間を減らすため）
- 最新のエントリが先頭にくる（時系列逆順）


## カテゴリタグ一覧
- `#納品`：納品物・データ形式・提出先に関する質問
- `#修正`：デザイン・テキスト・機能の修正依頼に関する質問
- `#費用`：見積もり・追加費用・支払いに関する質問
- `#スケジュール`：納期・工程・進捗に関する質問
- `#CMS操作`：CMS・管理画面の操作方法に関する質問
- `#その他`：上記に該当しない質問


## Q&A一覧

### YYYY-MM-DD
タグ：#カテゴリ
頻度：1回目
Q：[質問内容]
A：[回答内容]
出典：メール（[クライアント名または会社名]）
```
