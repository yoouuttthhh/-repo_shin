# CLAUDE.md
どのセミナーでも同じ構成で記録するための、再利用プロンプトです。
文字起こしテキストが手に入ったら、これをそのままClaudeに貼って使います。


## 使い方
1. セミナーの文字起こしテキストを用意する
2. 下記プロンプトの `[セミナー名]` と `[ここに文字起こしテキストを貼る]` を書き換える
3. Claudeに貼って実行する
4. 出力結果を `seminars/YYYY-MM-DD_セミナー名.md` として保存する


## プロンプト本文
```
以下は音声認識による文字起こしで、単語間に不自然なスペースが
入っている場合があります。まず自然な日本語文章に整形し、
その上でWhy/What/How形式に要約してください。

セミナー名：[セミナー名]

【Why（なぜ）】
- このセミナーが扱っている問題意識・背景・目的は何か

【What（なに）】
- 具体的に紹介された手段・フレームワーク・事例は何か
- 登壇者が伝えたかった核心的な内容は何か

【How（どのように）】
- 実際にどう実践するか、具体的な手順や事例
- 明日から自分の仕事にどう応用できそうか

最後に「今回の学び（1〜2行）」として、自分の言葉で
要点をまとめてください。

文字起こし:
[ここに文字起こしテキストを貼る]
```


## 出力フォーマット
```markdown
# [セミナー名]

日付：YYYY-MM-DD
登壇：[登壇者名・所属]

## Why（なぜ）
（内容）

## What（なに）
（内容）

## How（どのように）
（内容）

## 今回の学び
（1〜2行）
```
